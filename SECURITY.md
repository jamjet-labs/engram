# Security Policy

If you discover a security vulnerability in Engram, please report it privately to **security@jamjet.dev**. We will respond within 5 business days and coordinate disclosure with you.

Please do not file public GitHub issues for security reports.

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |
| < 0.1.0 | ❌        |
